## <span class="underline-blue"><i class="fa-solid fa-moon"></i> ANCANA</span>

##### <span style="font-weight:bold;">鬼野 ねね（Onino Nene）</span>
::: details
- 曾定期委托她摘花打工，是重复劳动获取素材的狠人。
- 因其照顾能力出色，头脑聪明，思维方式相同等理由，成为了的無馬专属秘书。
  - 在受任教导偿还自己过错（摘花）的<Anchor href="tarachan">タラちゃん</Anchor>的时候，無馬高度赞赏了她聪明伶俐的行动，并让她留在自己身边担任秘书。
  - 在刚当上秘书的时候，被無馬冠以【鬼之子】【里梅】的称呼。
- 现在隶属于無馬的私立组织<Anchor href="arcana">ARCANA</Anchor>，担任干部一职。对無馬宣誓效忠，有着为了無馬可以赌上性命也在所不辞的觉悟。
- 因無馬私人医生复职时“ねね你也来当私人医生”一言而任职私人医生。与無馬一样，同为<Anchor href="sosu">そす</Anchor>さん的弟子。
- 为了失去单眼的無馬，精选了包含备用在内共5只漂亮的眼球。
- 没有什么事情是不能对無馬说的，反之亦然，两人彼此完全信任。
- 無馬正在将其培养成与自己同等的存在，并打算将来把<Anchor href="arcana">ARCANA</Anchor>将给ねね。

{.list-double-space}
::: 

##### <span style="font-weight:bold;">月ノ島 ごう（Tsukinoshima Gou）</span>
::: details
- 交往密切的素材商。由于双方都对彼此慷慨大方，经常互相表达好意。
- 在無馬刚加入<Anchor href="allin">ALLIN</Anchor>时，帮忙收购素材，甚至欣然接下如违法倒卖枪支，为了帮派活动必要的便利店代理经营等，对于白市民来说风险较高的工作。好感度很高，甚至月ノ島在自己的电话名单里备注“喜欢”的程度。
- 在<Anchor href="arcana">ARCANA</Anchor>建立次日加入，将自己的商业经验贡献于無馬一直策划的地下生意。双方共同构建双赢的关系。
- 在他决定要成为黑市民时，作为对白市民身份的饯别礼，無馬送了他一套火箭筒装备，并见证了他的警察局袭击。
- 在無馬退出后加入<Anchor href="allin">ALLIN</Anchor>。是少数会告诉無馬ALLIN内情的人。
- 但由于在加入ALLIN时曾宣誓效忠ALLIN Boss，在ARCANA内则是稍退一步的位置。
- 在無馬与<Anchor href="allin">ALLIN</Anchor>缔结能量饮料的专卖契约时，宣言会以个人名义无偿提供所有素材。

{.list-double-space}

##### <span style="font-weight:bold;">上田さん（Uedasan）</span>
::: details
- 原<Anchor href="casino">赌场</Anchor>员工，负责活动事宜。为了成为修车行【[9055](# "修车行 店长为上田さん")】的社长而选择离职，是赌场的创业成员的一员，感情很深，现如今则作为鳥羽关系很好的交谈对象，一起为了赌场的繁盛而努力。
- 对無馬抱有尊敬之情的同时，也理解無馬不太懂白市民的娱乐，所以每次在城里相遇时就会尝试邀请無馬一起游玩，是与無馬保持良好距离感的友人。
- 对于無馬的玩笑和随意对待，总是大方地笑着应对，性格十分随和。
- 与<Anchor href="toba">鳥羽</Anchor>一起得知了<Anchor href="arcana">ARCANA</Anchor>的事情，但由于当时还去了警察的职业体验，苦恼了很久。与鳥羽多次交谈以后，感觉跟着無馬会很有趣，于是决定加入。
  - 作为ARCANA非公开，并完全隐藏身份的制作要员成员提供协助
  - 曾有一次误将ARCANA活动的报告发到赌场的群聊中，差点暴露身份，但在無馬和鳥羽的应对下成功化解。
- 作为赛车界举足轻重的人物，两人有时也会有相关的交集。

{.list-double-space}

##### <span style="font-weight:bold;">タラちゃん（Tarachan）</span>
::: details
- 刚来洛圣都不久就劫持了無馬作为人质并要求<Anchor href="allin">ALLIN</Anchor>支付赎金的半灰，因此事两人相识。
- 这之后作为惩罚被迫摘花的本人觉醒了制药的才能。因此，他对违禁药物及作为原材料的花有着浓厚的兴趣。
- 平时是个笨蛋，且每次见面都被当成玩具。但是一旦谈及生意，则会切换为其他人格。無馬称呼此人格为“Tarasan”（“Tara先生”，原文“タラさん”）。
- 無馬看中他的劳作才能与对商业的敏锐嗅觉，邀请他加入<Anchor href="arcana">ARCANA</Anchor>。他对無馬想到的“有趣的事情”有着超乎常人的兴趣，察觉到可以作为良好合作伙伴的关系，于是加入。
- 加入后，为了能公开进行药物精制，成立了帮派【<Anchor href="madara">麻陀罗组</Anchor>】。由于鬼野在無馬的指示后一直负责照顾，并协助其成立帮派，因此总是被無馬成为自己的『傀儡帮派』。
- 这之后虽然为了优先帮派活动，在ARCANA内退居一旁，但是对無馬的提议或请求总是毫不犹豫地答应，相当信任無馬。至今两人仍旧是很好的商业伙伴。

{.list-double-space}
:::

##### <span style="font-weight:bold;">香月 ろぎあ（Kouzuki Rogia）</span>
::: details
- [魔法少女咖啡](# "魔法少女主题咖啡店")的副店长兼，和果子咖啡店[よすが](# "专售日式点心的咖啡店")的店长。以及在<Anchor href="taxi">菖光亭</Anchor>，[狗咖](# "狗狗咖啡店")等饮食店工作过的接人待客的专家。
- 無馬看中其工作能力，邀请她加入<Anchor href="casino">赌场</Anchor>。也曾长期收到<Anchor href="toba">鳥羽</Anchor>的劝诱，是备受期待的人才。
- 这之后，由于<Anchor href="toba">鳥羽</Anchor>暂时回国，她开始负责赌场和無馬集团的管理。现在则是作为鳥羽的辅佐，二人一起共同支撑着赌场的运营工作。
- 無馬还是<Anchor href="allin">ALLIN</Anchor>的时候，帮助她与当时断绝友人关系的<Anchor href="heruan">ヘルアン</Anchor>修复了友谊。并因此为契机二人开始交往，她对此感激不尽。
- 除了她在赌场的工作表现以外，無馬还看中她有着能够收集黑白双方情报的生活方式，并察觉其中互赢的局面，进而邀请她加入<Anchor href="arcana">ARCANA</Anchor>。在宣言了自己会优先考虑男朋友的前提下后，为了报恩决定加入。其坚持本心的回答给無馬留下了良好的印象。
- 在第一次将她招待至<Anchor href="arcana">ARCANA</Anchor>基地时，其与平时打扮大相径庭的伪装让無馬和鬼野大吃一惊。
- 自从ARCANA从<Anchor href="allin">ALLIN</Anchor>独立后，优先考虑男友所属的ALLIN的香月处于稍微退居一旁的状态，但如今仍然通过鬼野，作为制作要员成员尽心尽力。

##### <span style="font-weight:bold;">麻林 ラル（Asabayashi Raru）</span>
::: details 
- 不管怎么看都像是黑市民的白市民。开了一家名为【[麻SICK RECORDS](# "饮食店兼唱片行 店长为麻林 ラル")】的唱片店，致力于为这座城市带来音乐。
- 不屈居人下主义。但本人以”只要是比自己聪明的人就可以“的理由，成为了部下。
- 出于对無馬的关心而行动的同时，也会作为地下事业同伙配合开展各种事业。
- 经常在自己的店里，制作出售一些名字和设计都很极限的商品。
- <Anchor href="arcana">ARCANA</Anchor>干部，担任面向黑社会的活动公司【WAND VOX】的负责人。

##### <span style="font-weight:bold;">大川（Ookawa）</span>
::: details
- 修车行【[A2 Lab](# "修车行 店长为大川")】的老板。经常委托他修理和改装车辆，以及向其打听新上市贩售的车辆情报等。
- 無馬在退出 · 逃离<Anchor href="allin">ALLIN</Anchor>的时候，依靠的机械师。
- 在無馬作为Doctor BONGORE行动的时候提供了很多帮助。很喜欢BONGORE姿势。
- 在烦恼生存方式的时候收到了無馬的劝诱，借着车辆协会的解体的时机，作为<Anchor href="arcana">ARCANA</Anchor>干部加入。负责车辆相关的事业和协调工作。

{.list-double-space}

##### <span style="font-weight:bold;">柳田 ライアン（Yanagita Raian）</span>
::: details
- 原帮派成员，曾在MOZU担任ヴァンさん的右腕。
  - 退出MOZU后仍就作为MOZU的专属私人医生。
- 在他利用过高利贷后，与其成为了称兄道弟的朋友。
- 两人共同的熟人是一位神秘的投资家“Cold Sirius <CollapsableText label="※">（曾关注过刚搬来洛圣都的紅一行人）</CollapsableText>”
- 在“帮派专属私人医生”制度废止时加入了<Anchor href="arcana">ARCANA</Anchor>。      
当初预定让他任职负责统括佣兵部队的干部，但由于他与老东家<Anchor href="mozu">MOZU</Anchor>之间的问题 <CollapsableText label="※">（在已经退出MOZU的情况下，拜托MOZU成员给自己调度枪支，并在被<Anchor href="van">ヴァン ダーマー</Anchor>问及原因的时候撒谎）</CollapsableText>，暂时只作为正式”成员“加入。
- 在無馬帮忙解决了上述的问题后，在洛圣都里，他除了宣誓效忠無馬以外，没有任何其他活路。至此他成为了对無馬言听计从的忠实部下。

##### <span style="font-weight:bold;">JD</span>
::: details
- 在Legion经营热狗店的青年。
- 将自己在找人的事情告诉了<Anchor href="ookawa">大川</Anchor>后，经由大川与無馬建立了联系。
- 在将自己的目的告诉無馬后，接受了来自無馬的劝诱，暂时作为佣兵成为<Anchor href="arcana">ARCANA</Anchor>的协力者。
- 之后，随着<Anchor href="raian">柳田 ライアン</Anchor>的降格和<Anchor href="jd">JD</Anchor>需求的“能获得更多情报的立场”一同，虽然是暂时的，但正式作为<Anchor href="arcana">ARCANA</Anchor>干部加入。
- 负责佣兵部队【<Anchor href="academy">ACADEMY</Anchor>】的汇总工作，但随着ACADEMY的解散，其作为干部的职责也结束了。同时，本人也在加入<Anchor href="yaroya">868</Anchor>后，逐渐退出了ARCANA的活动。此外，关于是否退出ARCANA，由于与無馬的忙碌期重叠，二人尚且为此进行谈话，现处于搁置状态。

##### <span style="font-weight:bold;">生雲丹 よづな（Namauni Yozuna）</span>
::: details
- 原【<Anchor href="iris">IRiS</Anchor>】成员。在帮派解散后被無馬雇用，并加入了<Anchor href="academy">ACADEMY</Anchor>。
- 本人自诩擅长听从指令，作为無馬的宠物1号~~被饲养~~。虽然有时言行不正，但正在被無馬慢慢矫正。
- 与其他男性一样称呼無馬为“daring”，但無馬本人不仅不介意，还对她采取十分冷淡和随意应付的态度，她似乎对此有些困惑。        
  無馬则是“只要能派上用场就够了”的态度，实际上似乎还是很喜欢生雲丹的。
- 随着<Anchor href="academy">ACADEMY</Anchor>的解散正式被雇用为<Anchor href="arcana">ARCANA</Anchor>的一员。从此以后，便仰慕着無馬并称呼其“Boss”。

##### <span style="font-weight:bold;">柳瀬 つん（Yanagi Tsun）</span>

- 与生雲丹同样，無馬也很中意他。
- 随着<Anchor href="academy">ACADEMY</Anchor>的解散正式被雇用为<Anchor href="arcana">ARCANA</Anchor>的一员。察觉到無馬优秀的枪法以及对部下的关心，现在有着相当强的忠诚心，仰慕着無馬并称呼其“Boss”。

##### <span style="font-weight:bold;">小峯 玲（Komine Rei）</span>
::: details
- 原任职于出租车公司【<Anchor href="taxi">菖光会</Anchor>】和帮派【<Anchor href="iris">IRiS</Anchor>】的中间管理层。曾是無馬就任出租车公司时入职负责人。
- 当初曾警惕無馬与其他帮派私通的可能。现在由于双方在各自帮派中处于相似的立场，成为经常互相聊天的朋友。
- 将無馬当作朋友。
  - 对無馬的印象是“喜欢天真无邪的恶作剧的人”。每次無馬起床时，会为了不让他感到无聊而寻找“玩具”。
- 在退出<Anchor href="iris">IRiS</Anchor>后，经常受到劝诱，加入了<Anchor href="academy">ACADEMY</Anchor>。
- 在<Anchor href="academy">ACADEMY</Anchor>解散后，正式加入<Anchor href="arcana">ARCANA</Anchor>。现在为了忙碌的無馬，负责收集和报告黑社会相关的情报。
  - 在考虑劝诱时，由于加入<Anchor href="arcana">ARCANA</Anchor>会导致无法加入帮派，無馬担心会让小峯感到无聊。对此，他表示「如果かなくん你需要我的话」，并加入。

##### <span style="font-weight:bold;">加藤 圧（Katou Atsu）</span>
::: details
- 表面上是汉堡店【[E5's BURGER](# "汉堡店 店长为加藤 圧")】的老板。暗地里则一手建立了麻药集团【<Anchor href="dep">DEP</Anchor>】并正在研发新药物。
- 虽然無馬仍是麻药集团的出资者和协助者，但是双方的信赖并不深。
- 曾一度以“没有好处”为由拒绝过加入<Anchor href="arcana">ARCANA</Anchor>的劝诱，但在迫于警察对<Anchor href="dep">DEP</Anchor>进行家宅搜查而请求ARCANA的护卫以及無馬的计策下，被迫在几乎没有退路的情况下坐上了谈判桌，在交谈中逐渐理解并共鸣無馬的商业理念后，最终正式加入了ARCANA。
- 一起商讨过关于曾为部下的<Anchor href="horon">高橋 滅論</Anchor>的事情，发现新型大型犯罪【金库抢劫】时与無馬和小峯一起享受犯罪的乐趣等等，察觉到自己与ARCANA成员们建立了比以前更深厚良好的关系的现在，时不时也会在<Anchor href="arcana">ARCANA</Anchor>无线电中露面。

##### <span style="font-weight:bold;">ぬくお（Nukuo）</span>

- 作为一位头脑聪明的人，经由小峯介绍而来。尽管时间不长，曾所属<Anchor href="academy">ACADEMY</Anchor>。
- 在<Anchor href="academy">ACADEMY</Anchor>解散后，仍然与其没有什么接触机会，但仍旧作为对<Anchor href="arcana">ARCANA</Anchor>有用的人才而被提及。起初，無馬评价他“太没有黑点了反而让人不安”，但实际面对面谈过后，认可了他的人品，最终正式加入。
- 现在他作为在<Anchor href="arcana">Arcana</Anchor>所有的事业中需要人才的时候能够提供辅助的全能手。
